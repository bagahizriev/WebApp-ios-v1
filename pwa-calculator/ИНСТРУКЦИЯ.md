# 📱 Подробная инструкция по установке PWA на iPhone

## Что такое PWA?

Progressive Web App (PWA) - это веб-приложение, которое работает как обычное мобильное приложение:

- Устанавливается на домашний экран
- Работает в полноэкранном режиме
- Работает офлайн
- Быстро загружается

## 🎯 Установка на iPhone (Safari)

### Шаг 1: Откройте приложение в Safari

⚠️ **Важно**: PWA на iPhone работает только через Safari, не через Chrome!

1. Откройте Safari на iPhone
2. Перейдите по адресу вашего приложения
3. Дождитесь полной загрузки страницы

### Шаг 2: Откройте меню "Поделиться"

1. Нажмите на кнопку **"Поделиться"** внизу экрана
    - Это квадратная иконка со стрелкой вверх
2. Появится меню с различными опциями

### Шаг 3: Добавьте на домашний экран

1. Прокрутите список опций вниз
2. Найдите и нажмите **"На экран «Домой»"** (Add to Home Screen)
    - Иконка выглядит как плюс в квадрате
3. Появится окно предпросмотра

### Шаг 4: Подтвердите установку

1. Вы увидите:
    - Название приложения (можно изменить)
    - Иконку приложения
    - URL адрес
2. Нажмите **"Добавить"** в правом верхнем углу

### Шаг 5: Готово!

✅ Приложение появится на домашнем экране iPhone
✅ Теперь можно запускать его как обычное приложение
✅ Оно будет работать даже без интернета

## 🔄 Как работает офлайн режим?

1. **Первое открытие**: Приложение загружает все необходимые файлы
2. **Service Worker**: Автоматически кеширует данные
3. **Последующие запуски**: Работает из кеша, даже без интернета
4. **IndexedDB**: Все ваши вычисления сохраняются локально

## 🎨 Что вы увидите после установки?

- **Иконка**: Фиолетовый градиент с эмодзи 📊
- **Название**: "Калькулятор Формул"
- **Полноэкранный режим**: Без адресной строки браузера
- **Быстрый запуск**: Мгновенная загрузка

## ❓ Частые вопросы

### Почему не работает в Chrome на iPhone?

Apple разрешает PWA только через Safari. В Chrome на iOS нет поддержки Service Worker.

### Как обновить приложение?

Просто откройте его - обновления загрузятся автоматически при наличии интернета.

### Как удалить приложение?

Как обычное приложение:

1. Долгое нажатие на иконку
2. Выберите "Удалить приложение"

### Занимает ли место на телефоне?

Да, но очень мало - около 1-2 МБ для кеша и данных.

### Будет ли работать без интернета?

Да! После первой загрузки приложение полностью работает офлайн.

## 🔧 Для разработчиков

### Проверка PWA на iPhone

1. Откройте Safari DevTools на Mac:
    - Safari → Настройки → Дополнения → Показать меню "Разработка"
    - Подключите iPhone по USB
    - Разработка → [Ваш iPhone] → [Страница]

2. Проверьте в консоли:

```javascript
// Service Worker активен?
navigator.serviceWorker.controller;

// IndexedDB работает?
indexedDB.databases();
```

### Требования для PWA на iOS

✅ HTTPS (или localhost для разработки)
✅ manifest.json с правильными полями
✅ Service Worker
✅ Иконки 192x192 и 512x512
✅ apple-touch-icon для iOS

### Особенности iOS

- Кеш Service Worker ограничен ~50 МБ
- Кеш очищается если приложение не используется 2+ недели
- Push-уведомления не поддерживаются (пока)
- Background sync не поддерживается

## 📊 Использование приложения

### Основные функции

1. **Ввод данных**:
    - Число A
    - Число B
    - Выбор операции

2. **Операции**:
    - Сложение: A + B
    - Умножение: A × B
    - Степень: A ^ B
    - Среднее: (A + B) / 2

3. **История**:
    - Автоматически сохраняется
    - Показывает дату и время
    - Можно очистить кнопкой

### Советы по использованию

💡 История сохраняется локально - не потеряется при закрытии
💡 Работает без интернета после первой загрузки
💡 Можно использовать десятичные числа (например, 3.14)
💡 История сортируется от новых к старым
